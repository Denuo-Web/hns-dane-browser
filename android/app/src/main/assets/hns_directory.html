Make My HNS Domain Work

Common fixes:
  • Direct site: add SYNTH4 or GLUE4 with a reachable address.
  • Delegated site: add NS plus usable GLUE4/GLUE6, then serve authoritative DNS on UDP/TCP 53.
  • Strict DANE: add HNS DS, DNSSEC-signed zone data, and TLSA for _443._tcp.name.
This directory is bundled locally with the app. It does not require network access to display.
