HNS DANE Browser

Enter a web address or Handshake name in the address bar to begin.

  • Open resolver trace to inspect HNS proofs and DNSSEC decisions.
  • Use the TLSA / DANE inspector to review certificate policy.
  • Enable Strict HNS mode to disable compatibility-resolver fallback.
